    **Part A: Auto-capture screenshot for every step (per D-01, TRACE-01)**

    Modify the `captureStepScreenshot` method (line ~815) to ALWAYS attempt a screenshot capture, not just when a tool trace has a screenshotId. The current logic only saves a screenshot if `traces.find(t => t.screenshotId)` finds one. Change it to:

    1. First, check if any trace already has a screenshot buffer (existing behavior) and save it if so.
    2. If NO trace has a screenshot, attempt to take a fresh screenshot using `instrumentedClient`. The InstrumentedMcpClient should have a method to call the MCP's screenshot tool directly. If the client has a `callTool` method, call it with `browser_take_screenshot` or the equivalent tool name. Wrap in try/catch -- if it fails (e.g., no browser session), return null as before.
    3. Update the method signature to accept `instrumentedClient: InstrumentedMcpClient` as parameter (it's already passed from `trackStepResult`).

    Concrete change to `captureStepScreenshot`:
    ```typescript
    private async captureStepScreenshot(
      runId: string,
      stepId: string,
      traces: ToolCallTrace[],
      instrumentedClient: InstrumentedMcpClient
    ): Promise<{ screenshotId: string | null; screenshotPath: string | null }> {
      try {
        // 1. Check existing trace screenshots
        const traceWithScreenshot = traces.find((trace) => trace.screenshotId);
        let screenshotBuffer: Buffer | undefined;

        if (traceWithScreenshot?.screenshotId) {
          screenshotBuffer = instrumentedClient.getScreenshot(traceWithScreenshot.screenshotId);
        }

        // 2. If no trace screenshot, attempt a fresh capture via MCP tool
        if (!screenshotBuffer) {
          try {
            const result = await instrumentedClient.callTool("browser_take_screenshot", {});
            if (result && result.type === "success") {
              // Extract base64 image from result content
              const imgContent = result.content?.find((c: any) => c.type === "image" || (c.type === "text" && c.text));
              if (imgContent) {
                const base64Data = imgContent.data || imgContent.text;
                if (base64Data && typeof base64Data === "string") {
                  screenshotBuffer = Buffer.from(base64Data, "base64");
                }
              }
            }
          } catch {
            // MCP may not support screenshot or browser not active -- silently skip
          }
        }

        if (!screenshotBuffer) {
          return { screenshotId: null, screenshotPath: null };
        }

        const screenshotId = await saveFileScreenshot(
          screenshotBuffer,
          runId,
          stepId,
          SCREENSHOT_DIR,
          traceWithScreenshot?.toolId
        );
        const screenshotPath = resolveScreenshotImagePath(runId, stepId, screenshotId, SCREENSHOT_DIR);
        return { screenshotId, screenshotPath };
      } catch (error) {
        this.logger.warn({ err: error, runId, stepId }, "No se pudo guardar screenshot de evidencia");
        return { screenshotId: null, screenshotPath: null };
      }
    }
    ```
