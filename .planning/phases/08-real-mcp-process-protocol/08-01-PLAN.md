**Step 3 — Rewrite McpProcessManager (per D-05, D-06, D-07):**
Replace the entire `McpProcessManager` class to:
1. `import { Client } from '@modelcontextprotocol/sdk/client'`
2. `import { StdioClientTransport } from '@modelcontextprotocol/sdk/client/stdio'`
3. `import { MCP_REGISTRY } from '../../shared/registry'`
4. `import type { BaseMcpClient, ToolResult } from './InstrumentedMcpClient'`
5. Class implements `BaseMcpClient`
6. Private fields: `transport: StdioClientTransport | null = null`, `client: Client | null = null`, `mcpId: string`
7. Public fields preserved: `pid: number | null`, `startedAt: Date | null`, `crashed: boolean`, `crashReason: string | null`
8. Add `private tools: Array<{ name: string; description?: string; inputSchema?: Record<string, unknown> }> = []`

**Constructor:** Takes `mcpId: string`. Validates `MCP_REGISTRY[mcpId]?.spawnCommand` exists and has length > 0, throws `Error(\`No spawnCommand in registry for \${mcpId}\`)` if missing.

**spawn():** Returns `Promise<{ pid: number; startedAt: Date }>`.
- Throws if `this.client !== null` ("MCP process already running")
- Reads `const [command, ...args] = MCP_REGISTRY[this.mcpId].spawnCommand!`
- Creates `new StdioClientTransport({ command, args, stderr: 'pipe' })`
- Creates `new Client({ name: 'mcp-bench', version: '1.0.0' }, { capabilities: {} })`
- Wires `transport.onclose` and `transport.onerror` to set `this.crashed = true` and `this.crashReason`
- Calls `await this.client.connect(this.transport)` — this spawns the process AND performs the initialize handshake (D-06)
- After connect: `this.pid = (this.transport as any).pid ?? null` (pid available after start)
- `this.startedAt = new Date()`
- `this.crashed = false` (reset after successful connect)
- Calls `const { tools } = await this.client.listTools()` and stores in `this.tools`
- Returns `{ pid: this.pid!, startedAt: this.startedAt }`

**healthCheck():** Returns `Promise<boolean>`. Returns `this.client !== null && !this.crashed`.

**callTool(name, args):** Returns `Promise<ToolResult>`.
- Throws if `!this.client` ("MCP client not initialized")
- `const result = await this.client.callTool({ name, arguments: args })`
- Maps to ToolResult:
  ```typescript
  return {
    type: result.isError ? 'error' : 'success',
    content: result.content as Array<{ type: string; text: string }>,
    error: result.isError ? String((result.content as any)?.[0]?.text ?? 'tool error') : undefined,
  };
  ```

**getTools():** Returns the stored tools array (public method).

**stop(timeoutMs = 5000):** Calls `await this.transport?.close()` wrapped in a `Promise.race` with a timeout. If timeout fires, attempt `process.kill(this.pid!, 'SIGKILL')` as fallback.

**dispose():** In a try/finally: calls `await this.stop(1000)` in try, then nulls `this.transport`, `this.client`, `this.pid`, `this.startedAt` in finally.

IMPORTANT: Do NOT call `transport.start()` manually — `client.connect(transport)` calls it internally. Calling both throws "StdioClientTransport already started!".
